# Members Dashboard

A WordPress plugin for managing member profiles, capturing moments, and QR code generation.

## Features

- Member profile management
- Moment capture functionality
- QR code generation and invitations
- Gravity Forms integration
- Screenshot capabilities

## Installation

1. Clone or download this repository
2. Place the folder in your WordPress plugins directory
3. Activate the plugin from the WordPress admin panel

## Usage

Use the provided shortcodes and functions to manage member data and moments on your site.
